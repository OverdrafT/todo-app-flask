# ToDo service
Flask service for CRUD operations for To-Do items.
## Here is a live demo of the app
[Flask ToDo app](http://95.217.158.152/)
## Prerequisites
Python 3.8.6 \
Flask 2.0.1 \
SQLAlchemy 1.4.18 \
Jinja2 3.0.1
## Getting started
- Clone repository
- `flask run`
- open http://127.0.0.1:5000/ in browser

Enjoy!
